# Marketing Funnel Dashboard (Dash + Plotly)

A functional-but-fun marketing funnel dashboard built with **Python Dash** and **Plotly**.  
It includes:
- Funnel overview chart with stage-to-stage conversion rates
- KPI cards with help tooltips
- Employee segmentation (dummy data)
- KPI detail page with a 3-month trend chart + mini table

---

## Demo Features
- **Overview**
  - Funnel bar chart (rounded corners)
  - KPI panel (click a KPI to open details)
